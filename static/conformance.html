  • Form attribute, can't be nested within other

Regarding embedded content (4.7):

  • Plugins and supported file formats vary between browsers. For example, Java source code displays correctly in Chrome, but not in Firefox.
  • Chrome supports WAV files, while Firefox requires Quicktime to run.

Regarding links (4.8):

  • For some reason, Chrome accepts the link to custom stylesheets, but Firefox doesn't. We have a custom stylesheet linked on our Links page, but Firefox doesn't pick it up. When we open the document inspector and import the style sheet directly, we get the expected result. Quite curious, indeed.
